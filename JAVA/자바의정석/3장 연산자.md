## 3장 연산자
* 연산을 수행하는 기호

### 연산자
#### 연산자와 피연산자
* 연산자(operator)  - 연산을 수행하는 기호(+, -, *, / 등)
* 피연산자(operand) - 연산자의 작업 대상(변수, 상수, 리터럴, 수식)

#### 연산자의 종류
* 산술 연산자 (+, -, *, /, %, <<, >>)
* 비교 연산자 (>, <, >-, <=, ==, !==)
* 논리 연산자 (&&, ||, !, &, |, ^, ~)
* 대입 연산자 (=)
* 기타 (type) ?: instanceof - 형변환 연산자, 삼항 연산자, instanceof연산자
